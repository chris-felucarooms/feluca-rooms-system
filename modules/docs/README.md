# Modulo Documentazione

Questo modulo raccoglie la **documentazione strutturata del sistema operativo Feluca Rooms**.

## Contenuti principali

- **Manuali operativi** dei workflow
- **Linee guida** per lo staff (standard, policy, best practice)
- **Glossario** dei termini usati nel sistema
- **Procedure di onboarding** per nuovi membri del team
- **Note di progettazione** e logica dei moduli

La documentazione qui descrive **come funziona il sistema**, mentre gli altri moduli (automations, templates, ecc.) contengono gli elementi operativi collegati.
